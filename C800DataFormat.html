SMW's Level block format

In SMW's levels, RAM address $C800 in banks $7E/$7F ($40/$41 in SA-1) contains all the block data of layer 1 and 2. They hold the map 16 numbers for every tile in the current level with the low bytes in bank $7E ($40 SA-1) and high bytes in bank $7F ($41 SA-1). The format of the blocks as follows:
